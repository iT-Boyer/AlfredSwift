Alfred Swift
---

升级，支持swift5最新版本

扩展 支持SPM集成到项目中
```
.package(name:"AlfredSwift", url: "https://github.com/iT-Boyer/AlfredSwift.git", from: "1.0.0")
```
使用：
```
import Alfred
```
